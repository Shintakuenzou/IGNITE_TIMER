# IGNITE_TIMER

O projeto DT Money é uma aplicação de gerenciamento financeiro desenvolvida para auxiliar os usuários no controle de suas finanças pessoais de forma simples e eficiente. 
Esta aplicação web moderna e amigável permite que os usuários registrem suas receitas e despesas, categorizem suas transações e acompanhem seu fluxo de caixa ao longo do tempo.

## Recursos Principais
-**Registro de Receitas e Despesas:** Os usuários podem facilmente adicionar suas transações financeiras, indicando se são receitas ou despesas.
-**Pesquisa de Despesas:** Os usuários podem pesquisar por despesas cadastradas com base em critérios como nome facilitando a localização de transações.

## Getting Started
- Clone o repositório para o seu ambiente local.
- Configure o ambiente e as variáveis de ambiente necessárias.
- Instale as dependências do projeto utilizando npm ou yarn.
  ```
  npm install
  # or
  yarn install
  ```
- Execute o servidor backend e o frontend.
- Acesse a aplicação no seu navegador ```http://localhost:3000``` e comece a gerenciar suas finanças!
